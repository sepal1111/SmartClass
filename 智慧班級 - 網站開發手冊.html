1. 專案概述

「智慧班級」是一個前後端分離的 Web 應用程式，專為單一班級的教學環境設計。它透過 Node.js 執行檔在教師的電腦上運行一個本地伺服器，學生則可透過區域網路內的行動裝置連線參與互動。專案的核心目標是將班級管理、教學互動與日常作業管理整合在單一平台中。

2. 技術棧 (Tech Stack)

後端 (Backend)

  • 執行環境: Node.js (v18.x)
  • 框架: Express.js
  • 資料庫: better-sqlite3
  • 即時通訊: Socket.IO
  • 認證: JWT
  • 封裝: PKG

前端 (Frontend)

  • 框架: Vue.js 3 (Composition API)
  • 建置工具: Vite
  • 路由: Vue Router
  • 樣式: Tailwind CSS

3. 開發環境設定

  1. 安裝 Node.js v18： 建議使用 nvm 來管理版本 (nvm install 18 && nvm use 18)。
  2. 安裝依賴套件： 分別在 client 和 server 目錄下執行 npm install。
  3. 啟動後端： 在 server 目錄下執行 npm run dev。
  4. 啟動前端： 在 client 目錄下執行 npm run dev。

4. 核心邏輯

認證流程 (Authentication)

系統採用 JWT (JSON Web Tokens) 進行無狀態認證。登入成功後，後端簽發一個有時效性的 Token，前端將其儲存在 `localStorage`。之後的每次 API 請求，都會透過 `Authorization` 標頭將 Token 送回後端進行驗證。

4. 核心邏輯

即時通訊 (Socket.IO)

PingPong 和 QuizRace 功能高度依賴 Socket.IO。後端透過 Rooms 機制來隔離不同的活動房間，並透過 事件驅動 (Event-driven) 的方式與前端溝通。所有遊戲狀態都在後端管理，前端僅作為畫面的呈現與事件的觸發器。

4. 核心邏輯

封裝與部署 (Packaging)

透過根目錄的 build.bat (Windows) 或 build.sh (macOS) 腳本，使用 PKG 工具將整個 Node.js 後端應用打包成一個單一執行檔。腳本會自動處理前端建置、原生模組複製 (`.node` 檔案)，並產生一個方便啟動的快捷方式。
