🚀 程式啟動與資料管理

Release 資料夾結構

智慧教室/ │ ├── 啟動智慧班級.bat <-- ✅ 啟動執行檔 │ ├── bin/ <-- 核心程式區 ├── data/ <-- 核心資料庫 ├── photos/ <-- 學生照片 └── uploads/ <-- 學生繳交的作業

👉 只需雙擊 啟動智慧班級.bat 即可啟動服務。

⚠️ 程式運行期間，請勿關閉黑色命令視窗。

🔄 如何備份與還原？

備份資料

將 data, photos, uploads 這三個資料夾完整複製到一個安全的地方 (例如：隨身碟、雲端硬碟)。

還原資料

將您備份好的三個資料夾，直接複製回 智慧教室 資料夾中，覆蓋掉原有的檔案即可。

💡 跨電腦使用提示：要移動到其他電腦進行課程時，只需要將整個資料夾複製帶走即可延續教學。

💻 網站功能操作說明

系統主畫面: 登入後的主介面，包含導覽列與內容區。

學生管理: 手動或批次匯入學生資料，並上傳照片。

課堂儀表板: 隨機抽籤、表現加分、登記出缺席的核心功能區。

座位表設定: 拖拉學生、調整版面，打造您的專屬座位表。

成績管理: 記錄課堂表現與各項評量分數。

作業管理: 建立數位作業，查看學生繳交狀況。

PingPong 即時問答: 快速匿名的課堂即時反饋工具。

QuizRace 搶答競賽: 將課堂測驗變成一場刺激的遊戲！

🧑‍🎓 學生管理

手動新增/編輯

點擊按鈕，即可在彈出視窗中輸入或修改學生的學號、姓名、帳號、密碼等資料。

批次匯入學生

從 Excel (.xlsx) 或 CSV (.csv) 檔案匯入。欄位需包含：學號, 姓名, 班級, 座號等。

批次上傳照片

選擇多張照片檔案。為了讓系統能自動對應，請務必將照片檔名命名為 學號.jpg (例如：109001.jpg)。

📊 課堂儀表板

隨機抽籤：從「出席」學生中隨機抽取一位。

課堂表現加減分：點擊學生座位即可為他 +1 或 -1 分。

批次登記出缺席：快速設定全班的出缺席狀態。

學期統計：查看指定區間內全班的出缺席統計與出席率。

🪑 座位表設定

拖曳學生：從左側學生列表拖曳至右側座位區。

調整版面：自由增減座位的行數與列數。

自動排列：系統根據座號自動依序排入座位。

⚠️ 完成編排後，務必點擊「儲存座位表」按鈕。

📝 成績管理 & 📁 作業管理

成績管理

用於記錄學生的各項評量分數。可登錄新成績、總覽與修改，並設定科目。

作業管理

建立數位作業項目，設定截止日期，並可線上查看學生繳交的檔案。

🙋 PingPong 即時問答

一個快速、匿名的課堂即時反饋工具，適合用於快速提問與收集全班意見。

匿名參與：學生只需輸入暱稱即可加入，無需登入。

多元題型：支援單選、是非、簡答、繪圖等題型。

即時結果：作答結束後，立即以圖表或縮圖牆呈現全班結果。

🏆 QuizRace 搶答競賽

1. 建立題庫

進入「搶答競賽」頁面，建立測驗集並管理題目。支援手動新增或從 Excel 批次匯入。

2. 主持遊戲

點擊「開始競賽」，在遊戲大廳等待學生加入。全員到齊後即可開始刺激的問答競賽！

3. 學生參與

學生掃描 QR Code 或輸入房間代碼即可加入，無需登入，準備搶答！

Excel 匯入格式提醒

欄位需依序為：
題目文字, 正確答案, 選項1, 選項2, 選項3, 選項4
